???+ note "표 생성 가이드"

    표는 문서 내 **지정된 위치에 새로운 표를 생성**하는 기능입니다.  
    2차원 배열 데이터를 기반으로 **행/열 구성, 병합, 스타일**을 모두 제어할 수 있습니다.

    ---

    **기본 규칙**

    - 표는 `elements` 배열 내부에서 **`type: "table"`** 로 선언합니다.
    - `data`는 **2차원 배열 형태**로 전달해야 합니다.
    - `data`의 첫 번째 행은 **칼럼명(헤더 행)**으로 사용됩니다.
    - 스타일 적용 시 `style` 객체를 선택적으로 사용할 수 있습니다.

    ---

    **사용 예시**

    ```json title="table create example"hl_lines="3"
    {
      "placeholder": "table_3",
      "type": "table",
      "data": [
        ["구분", "항목", "금액"],
        ["1", "품목A", "100,000"],
        ["2", "품목C", "200,000"],
        ["3", "품목D", "300,000"]
      ]
    }
    ```
