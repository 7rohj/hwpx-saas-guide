# AutoHWPX 사용자 가이드 
(260312 ...작성중...)

```이 문서는 AutoHWPX 를 처음 사용하는 사용자를 위한 가이드입니다.``` </br>

AutoHWPX는 **템플릿과 JSON 데이터를 결합하여 문서를 자동 생성하는 서비스**입니다. </br>
사용자는 `JSON 형식`의 요청 데이터를 통해 문서 생성에 필요한 `텍스트, 표, 스타일` 등의 요소를 정의 가능합니다.

> 본 가이드는 [AutoHWPX-API](https://api.autohwpx.com) 의 </br>
> **Generate API** `(/api/v1/generate)` **요청 (Request Body)** 에 </br>
> 사용되는 JSON 구조와 작성 방법을 제공합니다.

해당 JSON 형식은 향후 제공될 CLI 도구에서도 동일한 방식으로 사용될 예정입니다.
 
서비스 이용 중 궁금한 사항이 있을 경우 "문의남기기" 게시판을 통해 문의주세요.

AutoHWPX를 이용해 주셔서 감사합니다.
